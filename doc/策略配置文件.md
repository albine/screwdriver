# 实盘交易配置文件
# 格式: 股票代码,策略名称
#
# 股票代码格式（后缀可选，系统会自动识别）：
#   - 带后缀: 600759.SH, 002603.SZ（推荐，更明确）
#   - 不带后缀: 600759, 002603（自动识别：6开头→.SH, 其他→.SZ）
#
# 可用策略:
#   - PriceLevelVolumeStrategy: 价格档位挂单量监控策略
#   - TestStrategy: 测试策略（验证订单簿正确性）
#   - PerformanceStrategy: 性能测试策略
#   - PrintStrategy: 简单打印策略

# 示例配置（两种格式都可以）
002603.SZ,PrintStrategy              # 带后缀
600759,PriceLevelVolumeStrategy      # 不带后缀（自动识别为.SH）
